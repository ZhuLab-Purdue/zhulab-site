Add Data: Create a folder in database/ (e.g., MoS2) and add your files (summary.txt, properties.csv, images).

Update Site: Run "p"ython build_database.py".

Preview: Run "quarto preview" to check it.

Run "git add .", "git commit -m "xxx"", "git push"

Go Live: Run the publish command "quarto publish gh-pages".
